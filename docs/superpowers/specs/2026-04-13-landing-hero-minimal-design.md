# Landing Hero Minimal Cleanup

Date: 2026-04-13

## Goal

Make the left hero area of the landing page feel minimal, professional, and modern instead of loud or cluttered.

## Approved Direction

Use the `Executive minimal` direction.

## Scope

Only simplify the left blue hero panel on the landing page at `/`.

Do not change:
- phone verification behavior
- quick access links on the right panel
- logo usage
- app branding text
- install prompt logic

## Visual Changes

### Remove

- the `Phone-first access` pill
- the three informational cards:
  - `OTP`
  - `Daily`
  - `PWA`
- the lower trust/stat strip block

### Keep

- CISS logo
- `CISS Workforce`
- `Security workforce management platform`
- primary hero headline
- one supporting sentence

### Simplify

- reduce headline visual weight so it reads premium, not aggressive
- tighten spacing between logo lockup, headline, and support copy
- keep subtle background lighting, but reduce visual noise
- make the left panel feel like clean brand presentation, not a feature grid

## Final Left-Hero Structure

Top to bottom:

1. Logo lockup
2. Brand descriptor
3. One clean headline
4. One short supporting line

No pills. No stat cards. No secondary content blocks.

## Copy

Keep:

- `Security workforce management platform`
- `Verify guards, record attendance, and access admin operations fast.`

Headline can stay conceptually the same but should be visually calmer and fit in fewer lines.

## Success Criteria

- left side feels premium and uncluttered
- page still looks branded
- right verification card remains the main interactive zone
- desktop layout looks balanced and more professional at first glance
